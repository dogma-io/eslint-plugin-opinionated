## Installation

**npm**

```bash
npm install eslint-plugin-opinionated
```

**yarn**

```bash
yarn add eslint-plugin-opinionated
```

## Documentation

Add the following to your [ESLint configuration file][eslint-config-url]:

```js
{
  "extends": ["plugin:opinionated/recommended"],
  "plugins": ["opinionated"]
}
```
